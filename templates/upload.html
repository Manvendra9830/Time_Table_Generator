Important Instructions for Input File

  • File Format: The input file should be in .xlsx format.
  • Sheet Names: Ensure that the sheet names are exactly as specified (e.g., room, timing, faculty, etc.).
  • Data Consistency: Ensure that all IDs (e.g., faculty_id, course_number, timing_id) are consistent across sheets.
  • No Missing Data: Ensure that there are no missing values in any of the required columns.

1. Rooms Information

Sheet Name: room

Columns:

  • number: Unique identifier for each room (e.g., R1, R2).
  • capacity: Maximum seating capacity of the room.

Example:

numbercapacity
R160
R260

2. Timings Information

Sheet Name: timing

Columns:

  • id: Unique identifier for each timing slot (e.g., A, B).
  • time: The specific time slot (e.g., Monday 9:00-9:55).

Example:

idtime
AMonday 9:00-9:55, Wednesday 11:00-11:55
BMonday 10:00-10:55, Wednesday 9:00-9:55

3. Faculty Information

Sheet Name: faculty

Columns:

  • id: Unique identifier for each faculty member (e.g., Al, Pr).
  • name: Full name of the faculty member.

Example:

idname
AlDr. Alka
PrDr. Priodyuti Pradhan

4. Faculty Availability

Sheet Name: faculty_availability

Columns:

  • faculty_id: Identifier of the faculty member (matches id in the faculty sheet).
  • timing_id: Identifier of the timing slot the faculty is available for (matches id in the timing sheet).

Example:

faculty_idtiming_id
AlA,P,Q,R
PrA,C,F

5. Courses Information

Sheet Name: course

Columns:

  • number: Unique identifier for each course (e.g., CS201).
  • name: Name of the course.
  • max_numb_of_students: Maximum number of students that can enroll in the course.

Example:

numbernamemax_numb_of_students
CS201Data Structures50
CS202DAA50

6. Course-Faculty Mapping

Sheet Name: course_faculty

Columns:

  • course_number: Identifier of the course (matches number in the course sheet).
  • faculty_number: Identifier of the faculty member teaching the course (matches id in the faculty sheet).

Example:

course_numberfaculty_number
CS201Pr
CS202Ra

7. Departments Information

Sheet Name: dept

Columns:

  • name: Name of the department (e.g., Batch22, Batch23).

Example:

name
Batch22
Batch23

8. Department-Course Mapping

Sheet Name: dept_course

Columns:

  • name: Name of the department (matches name in the dept sheet).
  • course_numb: Identifier of the course offered by the department (matches number in the course sheet).

Example:

namecourse_numb
Batch22CS201,CS202,CS210,LA206
Batch23MA001,ID110,EE101,ID141
